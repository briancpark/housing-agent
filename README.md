# Housing Agent

A South Bay Area housing search tool that scrapes active listings from Redfin, calculates mortgage estimates, and displays everything on an interactive map.

## Features

- **Redfin Scraper** — Fetches active listings across 10 South Bay cities (San Jose, Sunnyvale, Santa Clara, Mountain View, Cupertino, Milpitas, Campbell, Los Gatos, Saratoga, Palo Alto)
- **Interactive Map** — Leaflet.js web app with color-coded markers, clustering, and click-to-zoom
- **Mortgage Calculator** — Full payment breakdown with configurable down payment, interest rate, and loan term
- **Filtering & Sorting** — Filter by price, beds, baths, city, property type, and max monthly payment. Sort by price, monthly cost, $/sqft, days listed, or size
- **CSV + JSON Export** — Spreadsheet-friendly CSV and JSON for the web app

## Quick Start

```bash
# Set up environment
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Scrape listings
python scraper.py

# Launch the web app
cd web && python -m http.server 8000
# Open http://localhost:8000
```

## Scraper

`scraper.py` uses Redfin's `gis-csv` endpoint with hardcoded region IDs for each city. It fetches up to 350 listings per city, deduplicates by address, and enriches each listing with mortgage calculations.

**Default search parameters** (edit at the top of `scraper.py`):
- Price: $500K–$2M
- Beds: 2+
- Types: House, Condo, Townhouse

**Output columns**: address, city, zip, price, beds, baths, sqft, $/sqft, lot size, year built, property type, HOA, days on market, status, Redfin URL, lat/lng, down payment, monthly mortgage, monthly total, total interest

### Adding Cities

To add a new city, you need its Redfin region ID. Look up the city on Redfin, inspect the network requests, and find the `region_id` parameter. Add it to the `SOUTH_BAY_CITIES` dict in `scraper.py`.

## Mortgage Calculator

Standalone CLI tool for exploring mortgage scenarios:

```bash
# Quick mode — pass a price
python mortgage_calc.py 1200000

# Interactive mode — prompts for inputs
python mortgage_calc.py
```

Shows: monthly payment breakdown, amortization preview, affordability (28% rule), and scenario comparison across different down payments and interest rates.

## Web App

The map app (`web/index.html`) is a single static HTML file with no build step. It loads `data/listings.json` generated by the scraper.

**Map features**:
- Color-coded markers: green (<$800K), yellow ($800K–$1.2M), orange ($1.2M–$1.6M), red (>$1.6M)
- Marker clustering with smooth zoom
- Click any listing card to fly to its marker
- Popup with full mortgage breakdown and Redfin link
- Adjustable mortgage parameters (recalculates all listings in-place)
- Dark theme (CARTO dark basemap)

**Dependencies** (loaded via CDN, no install needed):
- Leaflet.js 1.9.4
- Leaflet.markercluster 1.5.3
- OpenStreetMap tiles via CARTO

## Mortgage Defaults

| Parameter | Default | Notes |
|---|---|---|
| Down payment | 20% | Avoids PMI |
| Interest rate | 6.5% | Update to current market rate |
| Loan term | 30 years | 15-year also supported |
| Property tax | 1.2%/yr | Santa Clara County average |
| Insurance | $1,800/yr | Rough estimate |

These are configurable in both `scraper.py` (for CSV output) and in the web app UI (for real-time recalculation).
